# Base de Datos DAW 2025-2026

Este repositorio contiene los materiales, ejercicios y proyectos de la asignatura de Base de Datos para el curso 2025-2026 del ciclo de Desarrollo de Aplicaciones Web (DAW).

## Estructura del Repositorio

El contenido está organizado en las siguientes carpetas principales:

### `Clases/`
Contiene el material didáctico organizado por trimestres. Actualmente incluye el **Segundo Trimestre**, abarcando temas como:
- Agrupamiento de registros.
- Vistas, usuarios y permisos.
- Integración de Python con MySQL.
- Desarrollo de aplicaciones web básicas con Flask.
- Proyectos web con PHP y MySQL (Login, CRUD).

### `Ejercicios/`
Contiene ejercicios prácticos y evaluaciones, incluyendo:
- Scripts de creación de bases de datos y tablas.
- Simulacros de examen final (SQL + Markdown).
- Scripts en Python y HTML para prácticas diversas.

## Requisitos

Para ejecutar los ejemplos y ejercicios de este repositorio, necesitarás:

- **Base de Datos**: MySQL Server.
- **Lenguajes**:
    - Python 3.x
    - PHP (para los proyectos de login y CRUD web)
- **Librerías de Python**:
    - `mysql-connector-python`: Para conectar Python con MySQL.
    - `flask`: Para los ejemplos de aplicaciones web en Python.

## Instalación de Dependencias (Python)

Puedes instalar las librerías necesarias ejecutando:

```bash
pip install mysql-connector-python flask
```

## Uso

1.  **SQL**: Ejecuta los scripts `.sql` en tu gestor de base de datos favorito (Workbench, DBeaver, línea de comandos) para crear las bases de datos y tablas necesarias.
2.  **Python**: Ejecuta los scripts `.py` desde la terminal. Asegúrate de configurar correctamente las credenciales de conexión a la base de datos en cada script.
3.  **PHP**: Despliega los archivos `.php` y `.html` en un servidor web compatible (como Apache/XAMPP) para probar las interfaces web.
