# Ejercicios

Esta carpeta contiene ejercicios variados y materiales de evaluación.

## Contenido Principal

- **Examen final.sql** y **examen final.md**: Contienen el código SQL y la explicación teórica/práctica de un examen final. Cubre temas como:
    - Creación de BBDD y tablas.
    - Claves foráneas (Foreign Keys).
    - Operaciones CRUD.
    - Vistas y Usuarios.
- **Simulacro.sql**: Ejercicio de simulacro de examen.
- **Ejercicio.py** y **Ejercicio.html**: Scripts de práctica.
