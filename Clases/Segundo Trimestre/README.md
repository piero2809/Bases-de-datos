# Segundo Trimestre

En esta sección se encuentran los ejercicios y materiales trabajados durante el segundo trimestre.

## Directorios y Temas

### `004-Agrupamiento de registros`
En este módulo se trabajan consultas que implican agrupación de datos y operaciones básicas con Python.
- **SQL**: `GROUP BY`, `ORDER BY`, Proyecciones.
- **Python**: Conexión básica con MySQL, adaptación de peticiones.

### `006`
Este directorio contiene temas más avanzados de gestión de bases de datos y desarrollo web inicial con Python (Flask).
- **SQL**: Creación de Vistas, Usuarios y asignación de Privilegios. `JOINs` (Composiciones).
- **Python/Flask**: Plantillas de conexión, uso de diccionarios en cursores, lanzamiento de web básica con Flask.

### `008-Proyects`
Colección de mini-proyectos para poner en práctica lo aprendido, utilizando principalmente PHP y MySQL.
- **001-Login**: Sistema de inicio de sesión de usuarios.
- **002-CRUD**: Sistema básico de gestión de datos (Crear, Leer, Actualizar, Borrar).
- **003-Panel de control**: Panel administrativo.
- **004-Supercrud**: Versión extendida del CRUD.
- **Tratamiento de datos**: Scripts relacionados con el manejo de información.
